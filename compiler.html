Custom Virtual Machine & Language Parser

Language: C (C11)

Concepts: Compilers, Virtual Machines, Parsing, Code Generation, Stack Architecture

I developed a full compiler pipeline and virtual machine for the PL/0 programming language, a simplified Pascal-like language commonly used in compiler education. This project simulates the complete lifecycle of program execution, transforming raw source code into executable instructions and running them on a custom-built stack machine. The system consists of three core components: a lexical analyzer, a recursive descent parser with integrated code generation, and a virtual machine that executes the resulting instructions. The lexical analyzer processes raw input and converts it into structured tokens while supporting comment handling and error detection. The parser validates program structure using recursive descent techniques and generates intermediate P-Code instructions representing executable logic. These instructions are then executed by a custom virtual machine that simulates a stack-based processor with its own instruction set, registers, and activation record management. This project demonstrates low-level programming concepts including memory management, symbol table design, static scoping, stack frame construction, and control flow execution. It also required implementing procedure calls, nested scopes, conditional branching, and loop handling, closely modeling the behavior of real-world compilers and runtime systems.
