Bubble Sort Algorithm Visualizer

Languauge: Python

Framework: Tkinter

Cocepts: GUI Development, Animation Logic, Object-Oriented Design, Algorithms

I developed an interactive graphical application that visualizes the Bubble Sort algorithm in real time using Python and Tkinter. The program displays a set of randomized values and animates the sorting process step-by-step, allowing users to observe how elements are compared and swapped during execution. The application was designed with a focus on responsiveness and smooth animation. Instead of blocking the interface with traditional delays, the visualization uses Tkinter’s event-driven scheduling system to update the display asynchronously. This allows the sorting logic and animation system to work together without freezing the graphical interface. The architecture follows an object-oriented approach that separates the user interface from the underlying data representation. Each sortable element is treated as an object with visual and positional properties, enabling smooth movement during swaps and improving maintainability of the codebase. This project demonstrates understanding of algorithm visualization, GUI development, animation timing, and event-driven programming, while reinforcing core sorting algorithm logic and state management techniques.
